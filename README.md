# Анализ реализации

## UI/UX

- Заменить `prompt()` на модальное окно.
- Добавить индикатор загрузки. Сейчас загрузка моментальная из-за локальных данных.
- Добавить возможность редактирования аннотаций.
- Желательно добавить возможно скрытия всех аннотаций, их сворачивание в иконку (аналогично фигме).
- Не всегда понятно к какому элементу страницы аннотация. Можно добавить маштабирование элемента аннотации (актуально для режима отображения одной иконки).
- Добавить комментирование в аннотациях.

## Качество кода и поддержка

- Нестандартное CSS свойство `zoom`, не поддерживается в Firefox, может вызывать проблемы с позиционированием. Можно заменить на scale.
- Можно добавить валидацию данных. Уверен что в некоторых местах опустил для скорости работы.
- Можно добавить обработку ошибок загрузки изображений.
- Большие документы могут вызвать проблемы с загрузкой. Фиксить по мере определения требований, возможно это не нужно из-за формата получаемых данных.
